# Pankaj Solanki | Full Stack Developer Portfolio

A premium, modern developer portfolio built with **React.js**, **Vite**, **Tailwind CSS**, and **Framer Motion**. Featuring a high-end dark theme, glassmorphism, and smooth interactive animations.

## 🚀 Features
- **Modern Hero Section**: Circular profile photo with animated rings and achievement badges.
- **Projects Showcase**: Premium project cards with tech tags and interactive hover states.
- **Interactive Skills**: Categorized skills with visual progress indicators.
- **Experience Timeline**: Professional journey with a clean vertical timeline.
- **Functional Contact Form**: Integrated with **EmailJS** for direct mailbox communication.
- **Responsive Design**: Flawless experience across desktop, tablet, and mobile devices.
- **Optimized for Deployment**: Includes environment variable support and clean repository structure.

## 🛠️ Tech Stack
-   **Frontend**: React.js, Tailwind CSS
-   **Animations**: Framer Motion
-   **Icons**: React Icons
-   **Forms**: EmailJS
-   **Build Tool**: Vite

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/PankajSolanki/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Setup Environment Variables
Create a `.env` file in the root directory and add your EmailJS keys:
```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### 4. Run Locally
```bash
npm run dev
```

### 5. Build for Production
```bash
npm run build
```
